# ERP Backend API Server

기업 내부 업무 프로세스를 관리하기 위한  
**ERP(Enterprise Resource Planning) 백엔드 REST API 서버**입니다.

실제 ERP 시스템의 구조를 참고하여  
**도메인 분리, 인증/보안, 확장 가능한 API 설계**를 중심으로 구현했습니다.

---

## Tech Stack

- Java 17
- Spring Boot
- Spring Security + JWT
- JPA (Hibernate)
- Gradle
- MariaDB / MySQL

---

## Key Features

- ERP 도메인 기반 백엔드 API 설계
- JWT 기반 인증 및 권한 관리
- Spring Security 필터를 통한 요청 검증
- 도메인 중심 패키지 구조 설계
- 공통 예외 처리 및 일관된 API 응답 구조

---

## Portfolio Highlights

- 단순 CRUD가 아닌 **실무 ERP 구조를 고려한 백엔드 설계**
- 인증·보안·예외 처리 등 실무 필수 요소 구현
- 도메인 분리를 통한 유지보수성과 확장성 확보
- GitHub 기반 형상 관리 및 프로젝트 운영 경험
